## Installation
Clone this repo then:
```npm i```
The code is reliant on an .env file with
```
VITE_GITHUB_TOKEN=

```
Place the .env in the evironment folder

## Usage
To use:
```npm start:dev```
Or go to this website https://candidate-search-85yq.onrender.com/
